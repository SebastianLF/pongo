    Overview

    Metronic is a responsive and multipurpose admin and frontend theme powered with Twitter Bootstrap 3.3.4 Framework for admin and backend applications. Metronic Frontend has a clean and intuitive minimalistic design which makes your next project look awesome and yet user friendly. Metronic comes with a huge collection of plugins and UI components and it works in all major web browsers, tablets and phones.

Beginning of Page

<!DOCTYPE html>

6. SASS Support

To top

Metronic Admin Themes CSS is powered by SASS(sass-lang.com). Required SASS versions is v3.4.13 or later. We recommend you to use Koala App Tool from koala-app.com to compile realtime the SASS files. So just install the Koala App Tool and add the Metronic SASS directory(theme/sass) to start compiling the SASS files.

When you compile the sass files do not compile includable sass files that their filename starts with underscore(e.g: _variables.scss, _header.scss). Only the master files can be compiled(all the scss files without underscore infront of their filenames). Refer the below master SASS files to be compiled to spesific paths. For more info please check sass/koala-config.json file which used as a main configuration file to comple the Metronic sass files.
